## Install

Ruby versiond `cr` is the reference implementation, always up-to-date.
```bash
gem install cr.rb
```

## Other useful dictionaries

You can use `cr -a user/repo` to add other non-default dictionaries, for example:

```bash
cr -a ccmywish/signal_dict
```
